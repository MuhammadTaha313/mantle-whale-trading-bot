## 🤖 MantleWhale AI Trading Bot

Autonomous AI trading agent for Mantle Network that detects whale movements and executes trades automatically.

## 🚀 Run

## 📊 Features
- Scans MNT, WETH, USDT, USDC, mETH tokens
- AI signal engine - BUY/SELL/HOLD
- Auto Stop Loss (-8%) & Take Profit (+15%)
- Real-time monitoring every 30 seconds
